# Skills

Repo-local skills are thin agent workflows backed by the canonical guides in
[`docs/`](../docs/README.md). Keep durable system knowledge in docs and use skills
for operational checklists, routing, and guardrails.

## Available Skills

- [production-ops](production-ops/SKILL.md): production access, deploy or rollback,
  service triage, Podman containers, systemd units, `journalctl`, `/internal/jobs`,
  and production database inspection.
